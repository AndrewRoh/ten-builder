# 무료 vs Pro 비교

> 이 레포의 모든 콘텐츠는 무료입니다.
> Pro 버전은 더 체계적으로 학습하고 싶은 분을 위한 옵션입니다.

## 비교

| | 무료 (이 레포) | Pro (Gumroad) |
|---|---|---|
| **플레이북** | 핵심 5개 | 15개 + 고급 패턴 |
| **템플릿** | 범용 CLAUDE.md | 프로젝트별 최적화 (Spring, Next.js, Python) |
| **치트시트** | 원페이저 | PDF 가이드 + 영상 해설 |
| **에피소드 코드** | ✅ 전체 공개 | ✅ 동일 |
| **업데이트** | GitHub Release 알림 | 이메일 즉시 알림 + 우선 접근 |
| **팀 도입** | - | 팀 온보딩 체크리스트 |
| **가격** | 무료 | $29-49 |

## Pro가 필요한 경우

- AI 코딩 도구를 **팀에 도입**하려는 테크 리드
- 프로젝트 유형별로 **최적화된 설정**이 필요한 시니어 개발자
- **체계적인 커리큘럼**으로 빠르게 익히고 싶은 분

## 무료로 충분한 경우

- 개인 프로젝트에 AI 코딩을 적용하려는 분
- 핵심 패턴만 빠르게 파악하고 싶은 분
- 에피소드 코드를 참고하려는 분
