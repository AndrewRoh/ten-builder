# 플레이북 38: AI 코딩 에이전트 비용 최적화

> 월 $200 넘게 쓰고 있다면, 이 플레이북으로 절반 이하로 줄일 수 있어요 — 실제 토큰 사용 패턴 분석 기반

## 소요 시간

30-45분 (초기 설정), 이후 매주 10분 모니터링

## 사전 준비

- AI 코딩 도구 1개 이상 사용 중 (Claude Code, Cursor, Copilot 등)
- 최근 1개월 사용량 데이터 접근 가능
- 터미널 기본 사용 가능

## Step 1: 현재 비용 구조 파악하기

비용 최적화의 첫 단계는 내가 얼마를 어디에 쓰고 있는지 정확히 아는 거예요.

### 주요 도구별 가격 비교 (2026년 4월 기준)

| 도구 | 무료 | 기본 요금제 | 프로 요금제 | 비고 |
|------|------|------------|------------|------|
| GitHub Copilot | 제한적 무료 | $10/월 | $39/월 (Business) | 자동완성 중심 |
| Cursor | 무료 티어 | $20/월 (Pro) | $60/월 (Pro+) | IDE 통합 |
| Claude Code | - | $20/월 (Pro) | $100/월 (Max 5x) | 터미널 에이전트 |
| Windsurf | 개인 무료 | $20/월 (Pro) | $60/월 (Teams) | IDE 통합 |
| Gemini CLI | 무료 | - | - | Google 계정만 필요 |
| Aider | 무료 (OSS) | API 비용만 | API 비용만 | 모델 자유 선택 |

### 비용 진단 체크리스트

```bash
# Claude Code 사용량 확인
claude usage

# 최근 세션의 토큰 사용량 확인
claude sessions list --recent 7d

# API 사용 시 Anthropic 대시보드 확인
open https://console.anthropic.com/usage
```

> 대부분의 개발자가 월 $50-200 범위에서 사용하지만, 대규모 리팩토링이나 디버깅 세션에서 하루 $15-40까지 올라갈 수 있어요.

## Step 2: 토큰 소비 패턴 분석

비용의 80%는 보통 20%의 작업에서 발생해요. 어떤 작업이 토큰을 많이 먹는지 파악하세요.

### 토큰 많이 쓰는 작업 TOP 5

| 순위 | 작업 유형 | 평균 토큰 | 비용 영향 |
|------|----------|----------|----------|
| 1 | 대규모 리팩토링 | 150K-300K/세션 | 매우 높음 |
| 2 | 컨텍스트 200K 초과 세션 | 200K+ 입력 | 높음 (2배 가격) |
| 3 | 모호한 프롬프트 → 반복 수정 | 50K-100K 낭비 | 중간 |
| 4 | MCP 서버 컨텍스트 | 20K-50K/호출 | 누적 높음 |
| 5 | thinking 토큰 과다 | 30K-80K/요청 | 중간 |

### 핵심 진단: 200K 임계값

Claude Code API 가격표에서 가장 중요한 숫자는 **200K 토큰**이에요. 이 임계값을 넘으면 입력 토큰 가격이 2배가 됩니다.

| 모델 | 200K 이하 입력 | 200K 초과 입력 | 출력 |
|------|--------------|--------------|------|
| Opus 4.6 | $5/MTok | $10/MTok | $25/MTok |
| Sonnet 4.6 | $3/MTok | $6/MTok | $15/MTok |
| Haiku 4.5 | $1/MTok | - | $5/MTok |

```bash
# 세션 토큰이 200K에 근접하면 /compact 실행
/compact

# 완전히 새로 시작할 때
/clear
```

## Step 3: 모델 라우팅 전략

모든 작업에 같은 모델을 쓸 필요 없어요. 태스크 복잡도에 따라 모델을 나누면 60-90% 비용을 절감할 수 있어요.

### 3단계 모델 라우팅

```
간단한 작업 (50-60% of tasks)
├── 타입 에러 수정, 변수 이름 변경, 포맷팅
├── 추천: Haiku 4.5 ($1/MTok) 또는 Gemini CLI (무료)
└── 예상 절감: 80-95%

중간 복잡도 (30-35% of tasks)
├── 기능 구현, 테스트 작성, 코드 리뷰
├── 추천: Sonnet 4.6 ($3/MTok)
└── 예상 절감: 40-60%

높은 복잡도 (5-15% of tasks)
├── 아키텍처 설계, 대규모 리팩토링, 복잡한 디버깅
├── 추천: Opus 4.6 ($5/MTok)
└── 예상 절감: 0% (여기선 아끼지 마세요)
```

### 실전 적용 방법

```bash
# Claude Code에서 모델 변경
/model sonnet

# 간단한 수정은 Haiku로
/model haiku

# 복잡한 작업만 Opus로
/model opus
```

**Cursor 사용자라면:** 설정에서 모델별 사용 한도를 설정할 수 있어요. 기본 작업은 fast 모드, 복잡한 작업만 slow 모드를 사용하세요.

## Step 4: 프롬프트 최적화

구체적인 프롬프트 하나가 모호한 프롬프트 세 번보다 저렴해요.

### 비효율적 vs 효율적 프롬프트

```bash
# 비효율적 (토큰 낭비)
claude "이 코드 좀 개선해줘"
# → 에이전트가 전체 파일 읽기 + 여러 차례 질문 = 50K+ 토큰

# 효율적 (한 번에 해결)
claude "src/auth.js에서 상수를 파일 상단으로 추출하고, catch 블록에 에러 로깅 추가해줘"
# → 정확한 지시 = 10K 토큰
```

### 프롬프트 비용 절감 패턴

| 패턴 | 방법 | 절감 효과 |
|------|------|----------|
| 파일 범위 한정 | `src/auth.js`처럼 구체적 파일 지정 | 30-50% |
| 작업 범위 한정 | "함수 X의 에러 처리 추가"처럼 명확한 범위 | 40-60% |
| 출력 형식 지정 | "변경된 부분만 diff로 보여줘" | 20-30% |
| 체크리스트 제공 | 할 일 목록을 미리 정리해서 전달 | 50-70% |

## Step 5: 컨텍스트 관리

컨텍스트가 쌓이면 모든 요청이 비싸져요. 주기적으로 정리하세요.

### CLAUDE.md 최적화

```markdown
# CLAUDE.md (비용 효율 버전)

## 프로젝트 개요
Node.js + TypeScript 백엔드. PostgreSQL DB.

## 주요 규칙 (5줄 이내)
- 에러는 AppError 클래스 사용
- API 응답은 /types/api.ts 타입 사용
- 테스트 필수: vitest 사용

## 디렉토리 구조 (핵심만)
src/
  routes/    # API 라우트
  services/  # 비즈니스 로직
  models/    # DB 모델
```

> CLAUDE.md가 200줄 넘으면 **매 세션마다** 토큰을 낭비하는 거예요. 핵심만 남기고 나머지는 별도 파일로 분리하세요.

### 세션 관리 루틴

```bash
# 1. 새 태스크 시작 시 → 깨끗한 세션으로
/clear

# 2. 긴 세션 중간에 → 컨텍스트 압축
/compact

# 3. 토큰 확인 습관
/status  # 현재 세션 토큰 확인

# 4. thinking 토큰 제어 (필요할 때만)
/thinking off  # 간단한 작업
/thinking on   # 복잡한 작업
```

## Step 6: 추천 비용 구성

월 예산별로 가장 효율적인 도구 조합을 정리했어요.

### 월 $0 — 무료 구성

```
Gemini CLI (무료) + Aider (OSS) + GitHub Copilot Free
- 일일 검색/수정: Gemini CLI
- 코드 작성: Aider + 무료 모델
- 자동완성: Copilot Free
```

### 월 $20-30 — 가성비 구성

```
Claude Code Pro ($20) + Gemini CLI (무료)
- 주력: Claude Code (Sonnet 중심)
- 간단한 작업: Gemini CLI
- 월 500회 이상 요청 가능
```

### 월 $50-80 — 균형 구성

```
Claude Code Pro ($20) + Cursor Pro ($20) + Copilot ($10)
- 에이전트 작업: Claude Code
- IDE 통합: Cursor
- 자동완성: Copilot
- 모델 라우팅으로 비용 분산
```

### 월 $100+ — 성능 우선 구성

```
Claude Code Max 5x ($100) 또는 Cursor Pro+ ($60) + Claude API
- 제한 없이 사용
- 복잡한 작업도 Opus로 자유롭게
- ROI: 시간당 생산성 2-3배면 충분히 회수
```
